기강 스터디 도구의 자연어 entry point. 사용자가 슬래시 명령을 외울 필요 없이 평소 말투로 기강 작업을 시작할 수 있게 해줌.

## 발동 조건

이 스킬은 다음과 같은 자연어가 들어왔을 때 작동:

| 사용자 표현 예시 | 라우팅 대상 |
|----------------|------------|
| "신규 멤버 환경 셋업", "기강 깔아줘", "기강 환경 초기화", "처음 셋업" | `/gigang-init` |
| "기강 뭐 있어", "기강 명령 목록", "기강 도움말" | 아래 명령 목록 출력 |

**기강 무관한 일반 작업에는 발동 금지** — description 이 기강 한정이지만 Claude 가 광범위 매칭 가능. 의심되면 사용자에게 물어보기.

## 절차

### 1. 의도 파악

사용자 발언에서 다음 중 하나로 분류:
- **init**: 환경 셋업 / 신규 PC / 처음 / install 관련
- **help**: 도움말 / 명령 목록 / 뭐 있어

애매하면 AskUserQuestion으로 확인 (옵션 제시).

### 2. 라우팅

#### init 인 경우

사용자에게 한 줄 안내:
> 기강 환경 셋업이군요. `/gigang-init` 실행하겠습니다 — 필수 패키지(git, gh, uv 등), 터미널 설정, cc alias, Claude Code 플러그인, gstack, 기강 폴더 점검까지 한 번에 처리.

그 다음 `/gigang-init` 슬래시 명령을 호출 (Skill 도구 또는 명시적으로 사용자에게 그 명령 입력 안내).

**주의**: 슬래시 명령 자동 호출이 환경에 따라 안 될 수 있음. 그 경우 사용자에게 직접 `/gigang-init` 입력 요청.

#### help 인 경우

다음 표 출력:

```
기강 명령 목록
─────────────────────────────────────────────
/gigang-init      신규 멤버 환경 셋업 (패키지, 셋팅, cc, 플러그인)
/gigang-help      사용법 문서 표시
/gigang-version   현재 버전 및 업데이트 상태
/gigang           이 자연어 라우터 — "기강 뭐 해줘" 같은 표현용
─────────────────────────────────────────────

자연어 표현 예시:
  "환경 셋업해줘"        → /gigang-init
  "기강 도움말"          → /gigang-help

repo: https://github.com/Gigang-ST/gigang-skills
```

### 3. 사용자 확인 (애매한 경우만)

사용자 발언이 명확하지 않으면 (예: "기강 좀..."):

```
AskUserQuestion:
  무엇을 하시겠어요?
  A) 신규 멤버 환경 셋업 (/gigang-init)
  B) 명령 목록 보기
```

## 발동 안 되어야 하는 케이스

- "review 해줘" — 일반 코드 리뷰. 기강 무관.
- "코드 짜줘" — 일반 작업.
- "강 어쩌고" — 단순 단어 매칭만으로 발동하지 말 것. 기강 맥락이 명확해야.

의심되면 사용자에게 "기강 스터디 관련 작업 맞으세요?" 짧게 확인.

## 흔한 실수

- 슬래시 명령 자동 호출이 안 될 때 사용자에게 명시적으로 입력 안내 안 함 → 사용자가 멈춤.
- description 너무 광범위해서 무관한 작업에서도 발동 → "기강", "셋업" 같은 단어만으로 트리거되지 않게 기강 맥락 확인 필수.
- help 출력 시 표 잘림 → 모노스페이스 코드 블록 사용.
