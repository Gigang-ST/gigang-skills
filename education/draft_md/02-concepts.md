# 핵심 용어와 개념

> AI 도구를 쓰면서 자주 듣게 될 용어들을 비유로 설명합니다.

## LLM (Large Language Model)

**비유:** 세상의 모든 책을 읽은 사람.
질문하면 읽은 내용을 바탕으로 답한다.
ChatGPT, Claude 모두 LLM이다.

## ChatGPT vs Claude

둘 다 LLM이지만 성격이 다르다.

| | ChatGPT (OpenAI) | Claude (Anthropic) |
|---|---|---|
| 강점 | 이미지 생성, 범용 | 긴 문서 처리, 코드, 안전성 |
| 컨텍스트 | 보통 (~32K 토큰) | 매우 김 (~200K 토큰) |
| 가격 | 유료 | 유료 (팀 플랜) |

기강 스킬은 Claude를 씁니다.

## 프롬프트 (Prompt)

**비유:** AI에게 내리는 업무 지시서.
잘 쓸수록 결과가 좋아진다.

나쁜 프롬프트: "보고서 써줘"
좋은 프롬프트: "3월 활동 데이터를 항목별로 비교한 1페이지 요약 보고서를 써줘. 표 포함."

## 토큰 (Token)

**비유:** AI가 쓰는 화폐.
AI가 읽고 쓰는 텍스트 단위. 요금이 토큰 수에 비례한다.
대략 한글 1글자 ≈ 1~2토큰.

토큰이 다 떨어지면 → 잠깐 멈춤 → 충전 후 `/resume`으로 재개

## 컨텍스트 윈도우 (Context Window)

**비유:** AI의 단기 기억.
현재 대화창에서 AI가 기억할 수 있는 최대 분량.
창을 닫거나 `/clear`하면 리셋된다.

컨텍스트가 꽉 차면 → `/compact`로 압축

| 들어가는 것 | 처리 | 나오는 것 |
|---|---|---|
| 지시문 + 첨부 파일 + 이전 대화 | 컨텍스트 윈도우 | AI 응답 |

## Agentic AI

**비유:** 심부름꾼.
"카페 가서 아메리카노 사와"라고 하면
카페 찾고, 주문하고, 결제하고, 가져오는 모든 단계를 혼자 한다.

사람이 지시 → AI가 여러 단계 자동 실행 → 결과 보고

| 단계 | 내용 |
|---|---|
| 1. 계획 | 할 일 목록 작성 |
| 2. 실행 | 파일 읽기·쓰기·검색 등 |
| 3. 확인 | 결과 검토, 필요 시 재시도 |
| 4. 보고 | 완료 결과 전달 |

## Claude Code

**비유:** 직접 손발이 되어 일하는 AI 비서. 말로 지시하면 파일을 열고, 정리하고, 검색까지 직접 해준다.

터미널(명령줄)에서 돌아가는 Agentic AI 도구.
파일을 읽고, 쓰고, 실행하고, 검색하는 등 실제 작업을 수행한다.

## MCP (Model Context Protocol)

**비유:** AI용 플러그인 규격.
AI에게 외부 도구를 연결하는 표준 방법.

| MCP 서버 | AI가 할 수 있는 것 |
|---|---|
| Playwright MCP | 웹브라우저 조작 |
| Notion MCP | Notion 문서 읽기/쓰기 |
| Gmail MCP | 메일 읽기/작성 |
