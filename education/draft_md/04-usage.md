# 기본 사용법 & 단축키

> Claude Code를 실행하고 명령을 내리는 방법을 설명합니다.

## Claude Code 실행

PowerShell을 열고:

```powershell
claude
```

`>` 프롬프트가 나타나면 준비 완료.

종료: `Ctrl+C` 두 번 또는 `exit` 입력

## 명령 입력 방법

자연어로 지시하면 된다. 완벽한 문장이 아니어도 된다.

```
보고서 초안 작성해줘
```
```
이 파일 읽고 요약해줘
```
```
오타 찾아서 고쳐줘
```

## 필수 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+C` | 실행 중인 작업 취소/중단 |
| `Ctrl+V` | 붙여넣기 (터미널 내) |
| `↑` (위 화살표) | 이전 명령어 불러오기 |

## 컨텍스트 관리 명령어

Claude Code 안에서 `/`로 시작하는 명령어:

| 명령어 | 언제 쓰는가 |
|--------|------------|
| `/clear` | 새 작업 시작할 때 (기억 초기화) |
| `/compact` | 대화가 길어져서 느려질 때 (압축) |
| `/context` | 지금 얼마나 썼는지 확인할 때 |
| `/usage` | 토큰 사용량 확인할 때 |
| `/resume` | 토큰 충전 후 이어서 할 때 |

## 토큰이 다 떨어지면?

AI가 "토큰 한도에 도달했습니다" 메시지를 보내면:
1. 잠깐 기다린다 (충전 시점은 플랜에 따라 다름 — 팀 관리자에게 문의)
2. Claude Code를 다시 열고 `/resume` 입력
3. 이전 작업을 이어서 진행

> 중요한 작업은 중간중간 "지금까지 한 것 파일로 저장해줘"라고 요청하는 습관을 들이자.

## 실습 과제

Claude Code를 열고 아래 명령을 직접 입력해보세요:

```
안녕, 나는 기강 러닝팀 멤버야.
오늘 처음 Claude Code를 시작했어.
내 이름과 팀명으로 자기소개 문장 3개 만들어줘.
```

결과를 확인하고, `/usage`로 토큰 사용량도 확인해보세요.
