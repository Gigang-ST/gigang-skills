# 왜 Agentic AI인가

> ChatGPT와 무엇이 다른지, 왜 지금 써야 하는지를 설명합니다.

## ChatGPT와 뭐가 다른가

ChatGPT 브라우저 채팅은 "대화" 도구다.
질문하면 답한다. 내가 복사해서 어딘가에 붙여야 한다.

Agentic AI는 "행동" 도구다.
"보고서 초안 써서 파일로 저장해줘" 라고 하면 스스로 파일을 만든다.

| | ChatGPT 브라우저 | Agentic AI (Claude Code) |
|---|---|---|
| 하는 일 | 텍스트로 답변 | 파일 생성·수정·실행 |
| 내가 할 일 | 복사·붙여넣기 | 지시만 |
| 작업 단위 | 1문답 | 여러 단계 자동 진행 |
| 기억 | 창 닫으면 리셋 | 프로젝트 파일에 저장 |

## 실제로 어떻게 쓰는가

업무 예시:

**보고서 작성**
```
"지난 달 데이터 엑셀 파일 읽어서
항목별 실적 요약표 만들고 보고서 초안 작성해줘"
```
→ AI가 파일 열고, 데이터 읽고, 표 만들고, 문서 저장

**메일 정리**
```
"이 메일 목록에서 긴급한 것만 골라서
회신 초안을 각각 작성해줘"
```
→ AI가 분류하고 초안 파일 만들어줌

**반복 작업**
```
"매주 월요일 회의록 형식으로
이번 주 할 일 목록 정리해줘"
```
→ 한 번 설정하면 반복 자동화 가능

## 4배 생산성이란

동시에 여러 작업을 AI에게 병렬로 맡길 수 있다.
내가 A 작업을 검토하는 동안 AI는 B, C, D를 진행한다.

혼자서 4명 분량의 일을 처리하는 것이 목표.

| 내가 하는 일 | AI에게 맡긴 일 |
|---|---|
| 작업 A 검토 | 작업 B 진행 중 |
| | 작업 C 진행 중 |
| | 작업 D 진행 중 |
