# 부록 E: MCP 서버 추가하기

> AI에게 외부 도구를 연결하는 방법입니다.

## MCP란? (복습)

**비유:** 스마트폰 앱 설치.
기본 AI에 새로운 기능을 추가하는 플러그인 방식.

MCP 추가 전: AI가 텍스트만 다룸
MCP 추가 후: AI가 브라우저 조작, 문서 작성, 메일 읽기 등 가능

## 추천 MCP

| MCP | 기능 | 용도 |
|-----|------|------|
| Playwright | 웹브라우저 조작 | 로그인 후 데이터 수집 |
| Notion | Notion 읽기/쓰기 | 문서 자동화 |
| Gmail | 메일 읽기/작성 | 메일 자동화 |

## Playwright MCP 설치 (가장 많이 씀)

Claude Code를 실행한 상태에서:
```
/install playwright
```

설치 완료 후 브라우저 자동화 명령을 내릴 수 있다.

## 사용 예시

Playwright MCP 설치 후:
```
브라우저 열고 [사이트 주소] 접속해줘.
내가 로그인하면 그 다음에 [작업] 해줘.
```
