# 부록 H: gigang-뭐했더라

> education 커리큘럼 작성 이후 추가된 기능입니다.
> Claude Code 세션에서 과거 작업 내역을 자연어로 조회합니다.

## 무엇인가?

gigang-skills를 설치하면 Claude Code의 모든 세션이 자동으로 로그로 기록됩니다.
`gigang-뭐했더라` 스킬은 그 로그를 읽어서 "어제 뭐했더라" 같은 질문에 답해줍니다.

## 발동 방법

Claude Code에서 자연어로 말하면 됩니다:

| 발화 예시 | 조회 범위 |
|-----------|-----------|
| `어제 뭐했더라` | 어제 하루 |
| `지난주에 뭐했지` | 지난 주 전체 |
| `이번주 정리해줘` | 이번 주 월~오늘 |
| `5/13에 뭐했더라` | 5월 13일 |
| `2026-05-13 작업 내역` | 2026-05-13 |
| `전체 어제 뭐했더라` | 모든 프로젝트의 어제 |

날짜를 말하지 않으면 최근 3일을 기본으로 조회합니다.

## 어떻게 동작하는가

1. 현재 프로젝트 이름을 감지 (폴더명 기준)
2. `~/.claude/logs/prompts/<프로젝트>/<날짜>.md` 로그 파일 읽기
3. 오늘 기록은 각 턴의 첫 줄만 표시 (가볍게)
4. 과거 기록은 전체 내용을 3-5줄로 요약 + 캐시 저장

## 캐시 동작

한 번 요약한 날짜는 `~/.claude/logs/prompts/<프로젝트>/YYMMDD_summary.md` 에 저장됩니다.
다음에 같은 날짜를 조회할 때는 캐시를 바로 보여줘서 빠릅니다.

## 로그 파일 위치

```
~/.claude/logs/prompts/
  gigang_skills/
    2026-05-14.md      ← 오늘 로그
    260513_summary.md  ← 5월 13일 요약 캐시
  gigang-client/
    2026-05-14.md
```

각 `.md` 파일에는 프롬프트와 AI 응답이 턴 단위로 저장됩니다.

## 활용 팁

**주간 회고**:
```
지난주에 뭐했지?
어떤 작업에 AI를 제일 많이 썼어?
```

**특정 날 기억 되살리기**:
```
5/8에 뭐했더라, 그날 어떤 파일 만들었지?
```

**전체 프로젝트 조회**:
```
전체 어제 뭐했더라
```
→ gigang_skills, gigang-client 등 모든 프로젝트의 어제 작업을 한 번에 정리
