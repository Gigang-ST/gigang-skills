# 부록 D: CLAUDE.md로 내 규칙 만들기

> Claude에게 항상 지키게 할 규칙을 파일로 설정하는 방법입니다.

## CLAUDE.md란?

**비유:** 신입 직원에게 주는 업무 지침서.
Claude Code를 실행하면 이 파일을 자동으로 읽고 규칙을 따른다.
매번 같은 말을 반복하지 않아도 된다.

## 어디에 만드는가?

두 곳에 만들 수 있다:

| 위치 | 범위 | 용도 |
|------|------|------|
| `C:\Users\내이름\.claude\CLAUDE.md` | 전체 (모든 프로젝트) | 내 개인 선호 |
| `C:\WORK\내프로젝트\CLAUDE.md` | 해당 프로젝트만 | 프로젝트 규칙 |

## 작성 예시

```markdown
# 내 업무 규칙

## 응답 스타일
- 한국어로 답변해줘
- 핵심부터 먼저 말하고 설명은 뒤에
- 불필요한 인사말 생략

## 보고서 형식
- 제목 + 요약 3줄 + 세부 내용
- 표는 항상 포함
- 결론 먼저, 근거 나중

## 주의사항
- 개인정보는 절대 파일에 저장하지 말 것
- 외부 사이트 자동 로그인 시도 금지
```

## 만드는 법

PowerShell에서:
```powershell
notepad "$env:USERPROFILE\.claude\CLAUDE.md"
```

규칙 작성 후 저장하면 다음 Claude Code 실행부터 바로 적용.
