# 부록 F: 업무 자동화 레시피

> 실제 업무에 바로 적용할 수 있는 Claude Code 명령 예시 모음입니다.

## 사용법

아래 명령을 Claude Code에 복사해서 붙여넣고,
`[ ]` 안의 내용을 실제 내용으로 바꿔서 사용하세요.

---

## 레시피 1: 회의록 → 액션 아이템 추출

```
아래 회의록에서 액션 아이템을 추출해줘.
형식: 담당자 | 할 일 | 마감일
담당자가 명확하지 않으면 "미정"으로 표시해줘.

[회의록 내용 붙여넣기]
```

---

## 레시피 2: 메일 우선순위 분류

```
아래 메일 목록을 읽고 우선순위를 분류해줘.
기준: 긴급(오늘 회신) / 중요(이번 주) / 참고(나중에)
각 메일에 분류 이유 한 줄 추가해줘.

[메일 목록 붙여넣기]
```

---

## 레시피 3: Excel 데이터 정제

```
[파일명.xlsx] 파일을 읽어서:
1. 빈 행 제거
2. 날짜 형식을 YYYY-MM-DD로 통일
3. 수치 열에 천 단위 쉼표 추가
4. 중복 행 제거
결과를 [파일명_정제.xlsx]로 저장해줘.
```

---

## 레시피 4: 보고서 초안 자동 작성

```
아래 데이터를 바탕으로 [월]월 활동 보고서 초안을 작성해줘.
독자: 팀 운영진
분량: A4 1페이지
포함 항목: 핵심 지표, 전월 대비, 이슈사항, 다음 달 계획

[데이터 붙여넣기]
```

---

## 레시피 5: 반복 작업 스크립트 생성

```
매일 [폴더 경로]의 파일을 [다른 폴더]로 복사하고
파일명에 오늘 날짜를 붙이는 작업을 자동화해줘.
Windows 작업 스케줄러로 매일 오전 9시에 실행할 수 있게 해줘.
```

---

## 레시피 6: 문서 번역 + 요약

```
아래 [영어/일본어] 문서를 한국어로 번역하고,
핵심 내용을 5줄로 요약해줘.
전문 용어는 원어를 괄호 안에 표기해줘.

[문서 내용 붙여넣기]
```
