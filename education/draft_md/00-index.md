# 기강 Agentic AI 교육

> 이 자료는 기강 스킬 멤버를 위한 Agentic AI 집중 교육 커리큘럼입니다.

## 교육 목적

AI 도구를 왜 써야 하는지, 어떻게 써야 하는지 익혀서
각자의 업무 영역에서 생산성을 높이는 것.

## 사전 준비사항

교육 참가자가 교육 전에 준비해야 할 것:
- [ ] 노트북 지참 (Windows 10/11)
- [ ] Anthropic 계정 생성 (claude.ai)
- [ ] GitHub 계정 생성 (github.com) — gigang-skills 설치 시 필요
- [ ] Windows Terminal 설치 — 명령줄 창 앱 (IT 담당자에게 요청 가능)
